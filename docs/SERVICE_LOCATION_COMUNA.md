# Ubicación del servicio — dirección y comuna

El campo comuna siempre es visible. Nunca se oculta. Solo cambia estado (readonly/manual).

Cambios en este flujo requieren **revisión de producto**, no solo técnica. Ver comentario `STABLE MODULE` en `frontend/src/screens/client/ServiceLocationScreen.js` y el componente `ServiceLocationComunaSection.jsx`.
